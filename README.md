# README #

Skeleton project for Native Node
### What is this repository for? ###

## Usage ##
* To be able to read data, database must be created using the SQL files in the models folder. 
* 1) "piefitdatabase"
* 2) "piefitdata"
* 3) "piefitselect", or "piefitinsertcorecard" (optional)
